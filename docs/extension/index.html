扩展设置 ​

本指南将帮助你管理和配置 Chaterm 的各项扩展功能。通过扩展设置，你可以自定义终端的使用体验，提升工作效率。

扩展可用扩展功能 ​

1. 自动补全 ​

自动补全功能可以在你输入命令时提供智能提示，帮助你更快速地完成命令输入。

开启后：命令输入过程中会显示建议的补全选项

关闭后：需要完整手动输入命令

2. Vim 编辑器 ​

为喜欢使用 Vim 的用户提供可视化的 Vim 编辑模式。

开启后：可以使用 Vim 快捷键和命令进行文本编辑

关闭后：使用标准的文本编辑模式

3. 全局 Alias ​

允许你设置和使用全局命令别名，简化常用命令的输入。

开启后：可以使用已配置的命令别名

关闭后：需要输入完整的命令

4. 全局语法高亮 ​

为命令行界面提供语法高亮功能，提高代码和命令的可读性。

开启后：命令、参数和输出将带有颜色标识

关闭后：所有文本均以默认颜色显示

注意事项 ​

所有扩展设置的修改将即时生效

你可以随时根据需要开启或关闭这些功能

建议根据个人使用习惯和工作需求来配置这些扩展
