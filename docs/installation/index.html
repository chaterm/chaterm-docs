安装指南 ​

本指南将帮助你完成 Chaterm 的安装和初始配置。整个过程分为三个简单的步骤。

第一步：下载并安装 ​

访问 Chaterm Github

根据你的操作系统选择对应的安装包：

Windows: chaterm-version.exe

macOS: chaterm-version.dmg

Linux: chaterm-version.AppImage

下载完成后运行安装程序

按照安装向导的提示完成安装

第二步：账号登录 ​

Chaterm 支持多种登录方式：

邮箱验证码登录 ​

点击"邮箱登录"

输入你的邮箱地址

获取并输入验证码

点击"登录"按钮

账号密码登录 ​

点击"账号登录"

输入用户名和密码

点击"登录"按钮

提示：首次登录请使用邮箱验证码登录，更加安全便捷。登录进去后在用户信息里面添加用户名和密码后可使用账号密码登录。

第三步：基础配置 ​

完成登录后，你需要进行一些基础配置才能开始使用：

AI 模型配置 ​

进入"设置 > AI 模型"

选择想要使用的 AI 模型

配置相关参数

测试相关连通性

详细配置说明请参考 设置- AI 设置

终端配置 ​

点击"添加主机"

填写主机连接信息

保存后点击连接

详细配置说明请参考 主机-资产管理

如果你在安装使用过程中遇到任何问题，请提交问题。
