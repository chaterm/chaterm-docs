用户信息 ​

在 Chaterm 中，您可以方便地查看和更新您的个人信息。本文将指导您如何进行这些操作。

访问个人信息页面 ​

操作步骤 ​

进入个人信息页面 ​

在左侧菜单栏中找到 User 图标

点击进入个人信息页面

修改个人信息 ​

在个人信息页面，找到下方的编辑按钮

点击编辑按钮开始修改

可编辑信息 ​

您可以更新以下个人信息：

名称（Name）

用户名（Username）

手机号码（Phone Number）

保存更改 ​

完成信息修改后，点击保存按钮确认更改

系统会自动更新您的个人信息
